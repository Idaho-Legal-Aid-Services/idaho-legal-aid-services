## Residual risks / unknowns

- The active-risk heuristics remain deterministic phrase matching; additional kinship terms or more indirect third-person descriptions may still need future coverage.
- `PolicyFilter` still has broad emergency keywords, so some deadline phrasings containing generic urgency words like `today` can still route through policy rather than the urgency override. RAUD-17 intentionally left policy precedence unchanged.
- The change was verified with targeted unit and Drupal-unit suites only; no live endpoint or promptfoo verification was run for this remediation.

## Rollback notes

- Roll back the shared heuristic helper together with the `SafetyClassifier`, `PreRoutingDecisionEngine`, and test changes as one unit.
- No schema changes, config changes, cache migrations, or content/data rewrites were introduced.

- Broader multilingual family-relation crisis/scam phrasing not explicitly added in this remediation
