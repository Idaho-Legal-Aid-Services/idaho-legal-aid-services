# RAUD-13 Injected Logger Hardening Remediation Report

- Date: 2026-03-10
- Prompt ID: `RAUD-13`
- Finding IDs: `L1`, `N-28`
- Prior status: Open / verified gap. `AnalyticsLogger` still used static
  `\Drupal::logger()` in three catch blocks (stats write failure, no-answer
  persistence failure, analytics cleanup failure), and `ConversationLogger`
  still used static `\Drupal::logger()` in three paths (exchange write
  failure, cleanup info count, cleanup failure). The services therefore
  depended on global logger state instead of injectable module-scoped logging.
- Post-change status: Fixed
- Verification level tag: `Verified by execution`
